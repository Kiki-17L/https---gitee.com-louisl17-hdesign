# handsome

名字叫 handsome，随便起的，好听。

## node 版本

16.20.2

## 下载依赖

```sh
npm install
```

## 编译和启动

```sh
npm run dev
```

## 编译打包项目

```sh
npm run build
```

## 项目结构

### src 目录

| 目录          | 备注                    |
| ------------- | ----------------------- |
| api           | 存放接口文件            |
| assets        | 存放静态资源，图片之类  |
| components    | 存放公用组件            |
| router        | 路由文件                |
| store         | vuex 状态管理（分模块） |
| utils         | 存放单独封装的工具模块  |
| views         | 存放页面                |
| permission.js | 权限路由管理模块        |

### mock 目录

暂时没啥用
